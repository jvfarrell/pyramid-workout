Pyramid Workout Helper App by Victor Farrell

Welcome to the Pyramid Workout App. It's a work in progress.

I started doing these pyramid workouts a while back. I found it annoying to track the 60 second rest timing and the current rep number in my head. I wanted to let my mind go blank and do the work, so I built this handy website to track it for me on my phone.

I've made some minor improvements along the way and I'm using this site to experiment with new features. If you have ideas let me know.
